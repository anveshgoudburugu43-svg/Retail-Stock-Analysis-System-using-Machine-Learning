
# 📈 Analysis of Stock in Retail Using Machine Learning

## 📌 Project Overview

**Analysis of Stock in Retail Using Machine Learning** is a machine learning-based project that predicts future retail sales using historical sales data. The system helps retailers optimize inventory, reduce overstocking and understocking, improve customer satisfaction, and increase profits through accurate sales forecasting.

The model is trained using sales data from **January 2019 to December 2019** and predicts sales for **January 2020**. The predicted results are then compared with actual January 2020 sales data to evaluate the model's performance.

## 📊 Existing System

Traditional inventory systems mainly focus on recording sales and stock levels. They do not predict future demand, leading to:

- Overstocking
- Understocking
- Poor inventory planning
- Missed sales opportunities
- Lower customer satisfaction

---

## 💡 Proposed System

The proposed system uses **Machine Learning (Ridge Regression)** to forecast future sales by analyzing:

- Historical sales data
- Seasonal trends
- Promotions
- Customer purchasing behavior

### Benefits

- Better inventory planning
- Reduced stock shortages
- Reduced excess inventory
- Increased profitability
- Improved customer satisfaction

---

## 📈 Model Used

**Ridge Regression**

Ridge Regression is a supervised machine learning algorithm used for predicting continuous values. It helps improve prediction accuracy by reducing overfitting through L2 regularization.

---

## 📊 Dataset

The model is trained using retail sales data from:

- January 2019 – December 2019

Prediction is performed for:

- January 2020

The predicted results are compared with actual January 2020 sales to evaluate model accuracy.
